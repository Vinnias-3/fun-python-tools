A collection of educational and fun Python tools built for students and learners.

---

## 📦 Tools Included

### 1. 🧪 Advanced Chemistry Lab
- Element lookup via PubChem API
- Reaction equations database (12 reactions)
- Compound search
- Molar mass calculator
- pH calculator
- Chemistry quiz mode

### 2. 📊 Periodic Table Explorer
- Interactive element lookup
- Element comparison
- Advanced search by density, mass, name
- Category overview
- Quiz mode with scoring

### 3. 🎤 Text to Audio
- Converts any text to speech using eSpeak-ng
- Works offline
- Simple and fast

---

## 🚀 Installation

```bash
# Install required packages
sudo apt install espeak-ng -y
pip install periodictable requests --break-system-packages

# Run any tool
python3 "chemistry lab.py"
python3 "periodic table.py"
python3 "text to audio.py"
